# Contributing to SelfOptScheduler

Thank you for your interest in contributing to SelfOptScheduler!

## How to Contribute

### Reporting Issues
1. Check if the issue already exists
2. Provide:
   - Arduino board type and version
   - SelfOptScheduler version
   - Complete error message
   - Minimal code to reproduce

### Suggesting Features
1. Check existing feature requests
2. Describe the use case
3. Explain the benefit
4. Provide examples if possible

### Code Contributions
1. Fork the repository
2. Create a feature branch
3. Write clear commit messages
4. Test your changes
5. Submit a Pull Request

## Development Setup

1. Clone the repository
2. Install Arduino IDE
3. Open examples to test
4. Modify src/ files

## Coding Standards

- Use meaningful variable names
- Add comments for complex logic
- Follow Arduino library conventions
- Test on multiple boards

## Questions?

Open an issue or contact the maintainer.
